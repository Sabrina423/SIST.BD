# SIST.BD
 Sistema de Monitoramento Climático e Ambiental 🌪️

Sistema desenvolvido para detectar riscos ambientais como enchentes, queimadas e deslizamentos.  
Utiliza alertas, notificações, dados climáticos e IA para proteger a população.

## Estrutura do Banco
- 13 tabelas
- Relacionamentos 1:N e 1:1
- Chaves estrangeiras bem definidas
- Script 100% compatível com Oracle
